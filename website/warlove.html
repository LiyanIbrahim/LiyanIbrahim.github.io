حرب/حب (War/Love)

web art



حرب/حب (War/Love) is an interactive web art experience that reimagines physical spaces as digital vessels for memory and cultural identity. Abstracting from both Mahmoud Darwish’s poetry and the Mahmoud Darwish Museum, the project transforms literary and spatial elements into an immersive, interactive experience.

  • UX/UI & Technical Focus
      1. Interaction Design – Developed a responsive and immersive web experience that adapts to user exploration.
      2. Prototyping & Wireframing – Designed user flows and interface structures to map out non-linear storytelling structures using Figma.
      3. Front-End Development – Built using HTML, CSS, and JavaScript, with dynamic UI elements to for interactivity.
      4. Design Principles – Focused on accessibility, usability, and digital storytelling to evoke emotion and engagement.
